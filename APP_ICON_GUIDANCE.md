# Habit Tracker App Icon Guidance

## Suggested concept (minimal, App Store-friendly)
- **Background:** soft blue-to-mint gradient or plain white.
- **Main symbol:** rounded checkmark inside a simple circle/ring.
- **Meaning:** completion + streak consistency.
- **Style:** flat, high-contrast, no small text, no photo realism.

## Composition tips
- Keep the main symbol centered and bold.
- Avoid thin lines that disappear at small sizes.
- Use 1–2 colors plus white for clarity.
- Test readability at 60x60 and 1024x1024.

## Asset catalog setup
- Keep using `AppIcon` set in Xcode asset catalog.
- Provide a **1024x1024** source image and let Xcode derive size variants.
- Ensure no transparency in final export for App Store submission.

## Quick starter palette
- Primary: `#2F80ED`
- Secondary: `#56CCF2`
- Accent checkmark: `#FFFFFF`

This guidance is intentionally lightweight so the icon can be finalized later without changing app code.
